# TrustGuard 

**TrustGuard** — це комплексна вебплатформа для автоматизованого виявлення дезінформації та фейкових новин, побудована на базі мікросервісної архітектури та алгоритмів машинного навчання.

## Основні можливості
* **Мультиформатний аналіз:** Перевірка новин із простого тексту, URL-посилань, документів (PDF, DOCX) та навіть зображень (розпізнавання тексту через Tesseract OCR).
* **ML-класифікація:** Визначення достовірності новини (Fake / Real) за допомогою моделі Random Forest та векторизації TF-IDF.
* **Генерація підсумків:** Автоматичне створення стислого екстрактивного підсумку з довгого тексту статті.
* **Перевірка джерел:** Пошук схожих згадок у мережі серед бази надійних ЗМІ (інтеграція з DuckDuckGo Search).
* **Користувацький дашборд:** Реєстрація, збереження історії перевірок та перегляд особистої статистики.

## Технологічний стек
Проєкт розділено на два основні мікросервіси (головний вебсервер та сервіс інтелектуальної обробки), які взаємодіють між собою асинхронно.

**Веброзробка та Архітектура:**
* **Бекенд:** C#, ASP.NET Core (Clean Architecture)
* **База даних:** PostgreSQL (Neon), Entity Framework Core
* **Фронтенд:** Razor Pages, Bootstrap, Chart.js

**Машинне навчання та NLP:**
* **Мова та Фреймворк:** Python, FastAPI
* **ML Бібліотеки:** scikit-learn (Random Forest, TfidfVectorizer), NLTK
* **Асинхронність:** Redis (брокер повідомлень), Celery (worker-процес)

**Інфраструктура:**
* Docker & Docker Compose
* Хмарне розгортання: Microsoft Azure (Ubuntu Linux VM)

## Інструкція з локального запуску

### 1. Підготовка системи
* Переконайтеся, що у системі увімкнено **WSL 2** (для користувачів Windows).
* Завантажте та запустіть **Docker Desktop**. Переконайтеся, що сервіс Docker активний.

### 2. Клонування проєкту
Відкрийте термінал у вашій робочій директорії та виконайте команду:

```bash
git clone [https://github.com/Sonja3333333333333/trust-guard-2.git](https://github.com/Sonja3333333333333/trust-guard-2.git)
```

### 3. Завантаження файлів ШІ
1. Скачайте **3 файли** за посиланням: [Google Диск](ВСТАВ_СЮДИ_ПОСИЛАННЯ_НА_ДИСК)
2. Перемістіть усі 3 скачані файли строго у папку сервісу ML:

```bash
TrustGuard.ML/
```

### 4. Конфігурація додатку
1. Відкрийте файл конфігурації веб-додатка:

```bash
TrustGuard.Web/appsettings.json
```

2. Знайдіть у тексті поле для паролю пошти:
```bash
"SenderPassword": ""
```
3. Вставте всередину лапок готовий пароль додатку, який вказано у **Додатку А** курсової роботи.
4. Збережіть і закрийте файл.

### 5. Збірка та запуск
Відкрийте термінал у корені проєкту (там, де лежить файл `docker-compose.yml`) та виконайте команду для автоматичного білду:

```bash
docker compose up -d --build
```

### 6. Локальні адреси для перевірки
Після того, як усі контейнери запустяться, переходьте за адресами:

* **Головний сайт (C# Web):** [http://localhost:8080](http://localhost:8080)
* **Аналітика API (FastAPI Swagger):** [http://localhost:8000/docs](http://localhost:8000/docs)
